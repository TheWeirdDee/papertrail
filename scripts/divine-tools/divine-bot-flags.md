# Divine Bot Flags

- `AUTO_SET_USERNAME=0|1` — whether to attempt username setup.
- `FRIEND_RESET=0|1` — remove previous saved state before starting.
- `IDLE_RETRY_MS` — how long to wait between idle retries.
- `SAFE_FEE_USTX` — minimum balance required before a wallet is selected.
- `BASE_FEE_USTX` — base tx fee used when broadcasting transactions.
