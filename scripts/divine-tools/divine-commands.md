# Divine Bot Commands

Use these script aliases from the repository root.

```bash
npm run bot
npm run divine:balance-check
npm run divine:username-sync
npm run divine:state-report
npm run divine:reset-state
npm run divine:health-check
```

## Command purpose

- `bot`: run the main divine friend-farmer.
- `balance-check`: review all loaded wallet balances.
- `username-sync`: scan known wallets for on-chain usernames.
- `state-report`: print stored bot state and eligibility summary.
- `reset-state`: remove saved state to start fresh.
- `health-check`: validate bot preconditions.
